# Pokémon API Consumer

## Sobre

Este projeto é uma aplicação web simples que consome a [PokeAPI](https://pokeapi.co/) para buscar informações e exibir imagens de Pokémon. Desenvolvido com HTML, CSS e JavaScript puro, ele oferece uma interface intuitiva onde o usuário pode digitar o nome de um Pokémon e visualizar seus dados básicos e sprite. O projeto demonstra o uso de requisições assíncronas (`fetch`) para interagir com APIs externas e manipular o DOM para apresentar os resultados de forma dinâmica.

## Requisitos

Para executar este projeto, você precisará de:

*   **Navegador Web:** Um navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.) com suporte a JavaScript.
*   **Conexão com a Internet:** Para acessar a PokeAPI.

## Como Iniciar

Siga os passos abaixo para configurar e executar o projeto localmente:

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/hemkdev/pokemon-api.git
    cd pokemon-api
    ```

2.  **Abra o arquivo `index.html`:**

    Simplesmente abra o arquivo `index.html` em seu navegador web. Você pode fazer isso clicando duas vezes no arquivo ou arrastando-o para a janela do navegador.

    ```bash
    # Exemplo para Linux/macOS
    open index.html
    # Exemplo para Windows
    start index.html
    ```

## Uso

Após abrir o `index.html` no navegador, você verá uma interface com um campo de entrada de texto e um botão. Para usar a aplicação:

1.  **Digite o nome de um Pokémon:** No campo de texto, insira o nome de um Pokémon (ex: `pikachu`, `charmander`, `bulbasaur`).
2.  **Clique em "Buscar Pokémon":** Pressione o botão para iniciar a busca.
3.  **Visualize os resultados:** As informações e a imagem do Pokémon serão exibidas na tela. Se o Pokémon não for encontrado, uma mensagem de erro será mostrada.
